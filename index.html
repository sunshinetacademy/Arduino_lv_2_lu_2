مراجعة المحاضرة السابقة

❓ RGB اختصار لإيه؟

Red, Green, Blue – ثلاثة ألوان أساسية داخل LED واحد

❓ كام LED جوه الـ RGB؟

3 شرائح مدمجة في علبة شفافة واحدة

❓ digitalWrite vs analogWrite؟

digitalWrite: تشغيل/إطفاء فقط
analogWrite: تحكم في السطوع (PWM)

❓ نطاق PWM؟

من 0 (مطفأ) إلى 255 (أقصى سطوع)

المشكلة والحل: map()

🎛️ البوتانشيومتر

يقرأ الأردوينو قيمة:

0 ⟷ 1023

💡 RGB / PWM

يحتاج قيمة:

0 ⟷ 255

كيف نحول من نطاق لنطاق؟ نستخدم دالة سحرية:

map(value, 0, 1023, 0, 255);

وظيفتها: تحويل الأرقام من مقياس لمقياس آخر تلقائياً

مخطط التوصيل

🌈 RGB LED

R → Pin D9
G → Pin D10
B → Pin D11
GND → GND

🎛️ Potentiometer

طرف 1 → 5V
الوسط → A0
طرف 3 → GND
⚠️ افصل الأردوينو عن الكمبيوتر أثناء التوصيل!

الكود 1: قراءة قيمة المقاومة

int potPin = A0;

void setup() {
  Serial.begin(9600);
}

void loop() {
  int value = analogRead(potPin);
  Serial.println(value);
  delay(100);
}

📊 ماذا نتوقع على Serial Monitor؟

0 → 120 → 350 → 600 → 900 → 1023

لف المقاومة ببطء وشاهد الأرقام تتغير!

الكود 2: استخدام map()

int potPin = A0;

void setup() {
  Serial.begin(9600);
}

void loop() {
  int value = analogRead(potPin);
  int brightness = map(value, 0, 1023, 0, 255);
  
  Serial.print("Raw: "); Serial.print(value);
  Serial.print(" | Mapped: "); Serial.println(brightness);
  delay(100);
}

الشرح: الدالة map() حولت القيمة من 0-1023 إلى 0-255 لتناسب نظام الـ PWM في الأردوينو.

الكود 3: التحكم في اللون الأحمر

int redPin = 9;
int potPin = A0;

void setup() {
  pinMode(redPin, OUTPUT);
}

void loop() {
  int value = analogRead(potPin);
  int brightness = map(value, 0, 1023, 0, 255);
  
  analogWrite(redPin, brightness);
}
🎛️ لف المقاومة
📊 map() تحسب السطوع
🔴 الأحمر يزيد/يقل!

نماذج وتجارب سريعة

نموذج 1: الأخضر

analogWrite(greenPin, brightness);

نموذج 2: الأزرق

analogWrite(bluePin, brightness);

نموذج 3: التحكم في الأبيض (الثلاثة معاً)

analogWrite(redPin, brightness);
analogWrite(greenPin, brightness);
analogWrite(bluePin, brightness);
